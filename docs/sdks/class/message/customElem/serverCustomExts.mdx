# Server CustomExts

## 功能介绍

:::info

:::


## s_10001

> 加群消息的CustomExts


| 字段名称    | 字段类型  | 描述 |
| ---------- | -------- | ---- |
| amount     | NSString | 被邀者 |
| fromName   | NSString | 邀请者 |
